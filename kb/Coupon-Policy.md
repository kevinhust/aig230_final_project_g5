# Coupon & Promotion Rules (优惠券规则 / Règles de Coupons / Reglas de Cupones)

Category: Policies
Language: en-zh-fr-es

---

## English

**Q: How do I apply a coupon code?**
A: Enter the code in the "Promo Code" box at checkout and click "Apply." The discount will be reflected in your order total before payment.

**Q: Can I use multiple coupons on one order?**
A: Only one coupon code per order. However, coupons can be combined with: free shipping offers, loyalty points, and sale items (unless coupon excludes sale items).

**Q: My coupon code isn't working. Why?**
A: Common reasons: 1) Code expired. 2) Minimum purchase not met. 3) Item excluded from promotion. 4) Already used (one-time codes). 5) Typo - check capitalization.

**Q: Can I apply a coupon after placing an order?**
A: Unfortunately, coupons cannot be applied after checkout. Please verify all discounts before confirming your order.

**Q: Do you have student/military/senior discounts?**
A: Yes! Students get 15% off with valid .edu email verification. Military and seniors (65+) get 10% off with ID verification. Contact support for verification.

## 中文 (Chinese)

**Q: 如何使用优惠券？**
A: 在结账页面的"优惠码"框中输入代码，点击"应用"。折扣将在付款前反映在订单总额中。

**Q: 一个订单可以使用多个优惠券吗？**
A: 每个订单只能使用一个优惠码。但是，优惠券可以与以下优惠叠加：免运费、积分、打折商品（除非优惠券排除打折商品）。

**Q: 优惠券不工作，为什么？**
A: 常见原因：1) 代码已过期。2) 未达到最低消费。3) 商品不在促销范围内。4) 已使用过（一次性代码）。5) 输入错误 - 检查大小写。

**Q: 下单后可以补用优惠券吗？**
A: 很遗憾，结账后无法补用优惠券。请在确认订单前核实所有折扣。

**Q: 有学生/军人/老人折扣吗？**
A: 有！学生用.edu邮箱验证可享85折。军人和65岁以上老人凭ID验证享9折。联系客服进行验证。

## Français (French)

**Q: Comment appliquer un code promo?**
A: Entrez le code dans la case "Code Promo" au moment du paiement et cliquez sur "Appliquer". La remise sera reflétée dans le total.

**Q: Mon code promo ne fonctionne pas. Pourquoi?**
A: Raisons courantes: 1) Code expiré. 2) Achat minimum non atteint. 3) Article exclu. 4) Déjà utilisé. 5) Erreur de frappe.

## Español (Spanish)

**Q: ¿Cómo aplico un código de cupón?**
A: Ingresa el código en la casilla "Código Promocional" en el pago y haz clic en "Aplicar". El descuento se reflejará en el total.

**Q: ¿Puedo usar múltiples cupones en un pedido?**
A: Solo un código por pedido. Sin embargo, los cupones se pueden combinar con: envío gratis, puntos de lealtad y artículos en oferta.
